
# AI README Generator

## Description

A Python-based tool that leverages Ollama and phi3:mini model to generate structured output from user prompts, which is then formatted into an editable Markdown README file. Runs in your console!

## Features

- Python programming language
- Ollama integration for AI capabilities
- phi3:mini model usage for text generation
- Markdown formatting output
